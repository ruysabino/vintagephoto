Gerador de Polaroid com metadados EXIF

Faça upload de uma foto, escolha quais tags de metadados deseja mostrar, gere a arte em estilo polaroid e baixe a imagem final em PNG.

JPEG costuma trazer mais EXIF. PNG e imagens reenviadas por apps podem não ter metadados.
Tudo acontece no navegador. A imagem não é enviada para servidor, o que facilita hospedar no GitHub Pages.
