# 📸 Polaroid Generator

Gerador de imagens estilo Polaroid com leitura de metadados EXIF, localização e efeitos retrô. Totalmente feito com auxílio de IA generativa.

## 🚀 Features

- Upload de imagem ou link do Google Photos
- Leitura de EXIF (data, câmera, ISO, etc.)
- Conversão de GPS para localização (OpenStreetMap / Google Maps)
- Modo WhatsApp (data via nome do arquivo)
- Presets visuais (Classic, Scrapbook, Cinema, Summer)
- Efeitos opcionais:
  - Sépia
  - Grain (desgaste)
  - Poeira
  - Carimbo retrô
- Download em PNG
