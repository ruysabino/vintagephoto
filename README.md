# 📸 Polaroid Generator 3.0

Gerador de imagens estilo Polaroid com leitura de metadados EXIF, localização e efeitos retrô.

## 🚀 Features

- Upload de imagem ou link do Google Photos
- Leitura de EXIF (data, câmera, ISO, etc.)
- Conversão de GPS para localização (OpenStreetMap / Google Maps)
- Modo WhatsApp (data via nome do arquivo)
- Presets visuais (Classic, Scrapbook, Cinema, Summer)
- Efeitos opcionais:
  - Sépia
  - Grain (desgaste)
  - Poeira
  - Carimbo retrô
- Download em PNG

## 🌐 Deploy (GitHub Pages)

1. Suba os arquivos para o repositório
2. Vá em **Settings → Pages**
3. Selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Salve
